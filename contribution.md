## 📌 Project Overview

### 🔍 Description

**GiftMyWay** is a responsive front-end web application that simulates an online gift store. It allows users to browse categorized gift items and is designed to evolve into a fully functional e-commerce platform with dynamic and personalized features.

---

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Deployment:** GitHub Pages
* **Storage:** localStorage (for cart & upcoming features)

---

## ✨ Current Features

* Categorized gifting options (Men, Women, Kids)
* Clean and responsive UI
* Product card layout
* Custom gift section (UI + partial functionality)
* Add to Cart functionality
* Search & Filter system

## ⚙️ Getting Started

### ✅ Prerequisites

* A modern web browser (Chrome recommended)
* Git installed

---

### 🚀 Setup Instructions

```bash
# Fork the repository

# Clone your fork
git clone https://github.com/<your-username>/GiftMyWay.git

# Navigate into project
cd GiftMyWay

# Run the project
Open index.html in your browser
```

---

## 🔄 Development Workflow

```bash
# Create a new branch
git checkout -b feature/your-feature-name

# Make changes
git add .

# Commit changes
git commit -m "feat: add your feature"

# Push to GitHub
git push origin feature/your-feature-name
```

Then open a **Pull Request 🚀**

## 📝 Contribution Guidelines
